##简单数独类

 - 产生随机数独

算法分四步：
 1. 随机一串1-9的字符串，放入中间的宫中
 2. 通过上下移动行来得到左右两个宫，通过左右移动列来得到上下两个宫
 3. 移动左右两个宫的列得到四个角的宫
 4. 随机交换某些行和列，避免规律性太强

- 解数独

解法一： 普通回溯，DFS，实现较为简单，此代码也是采取的这种方法。

解法二： DLX，跳舞链，方法比较复杂，但本质上也是回溯，具体实现可参考[Sudoku Solver][1]。
